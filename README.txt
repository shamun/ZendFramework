Instruction for layout fitting:
-------------------------------

  1) please follow the directory skeleton exactly how it is
  2) put your public css/js/img/contents in following path
      public/css
      public/js
      public/img
  3) put your html and layout in
      application/layouts/scripts/<new directory>/header and footer only
                                                 {{see example files called layout.phtml}}

      application/views/<new directory>/body of the html
